Package icecube.uhe.points

This package contains the objects to define the particle location in its progation in the Earth. The methods are provided to get/set length of the trajectory, the propagation distance along the trajectory, the distance from the earth center at the current location and so on. It also handles the propagation medium: the local mass density, the mass number, the charge number for examle, can be acquired by the methods this package supplies you.
